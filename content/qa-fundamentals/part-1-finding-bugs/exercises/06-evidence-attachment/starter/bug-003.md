# [Registration] Duplicate email returns 500 Internal Server Error

**Severity:** High
**Priority:** Medium

## Environment

- **Browser:** Chrome 130
- **OS:** macOS 14
- **Viewport:** 1440x900
- **URL:** https://learnstack-taskforge-web.onrender.com/register

## Steps to Reproduce

1. Navigate to `https://learnstack-taskforge-web.onrender.com/register`
2. Fill in all fields with valid data — use an email that already has an account (e.g. `bob@taskforge.io`)
3. Click **Create Account**

## Expected Behavior

A user-friendly inline error message: "An account with this email already exists. Try logging in."

## Actual Behavior

The page returns a white screen with "500 Internal Server Error". No user-friendly message is shown.
