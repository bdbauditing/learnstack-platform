# Grader Notes — Exercise 04

Validation is structural only — we check that each rewritten report has:
1. All required fields (non-empty)
2. At least 3 specific steps (with selector or URL)
3. Expected and actual at least 10 chars each
4. Environment all filled in

We do NOT check if the learner identified the same bug correctly — any plausible rewrite that is structurally valid passes. The learning objective is "write good repro steps", not "guess the right bug".
