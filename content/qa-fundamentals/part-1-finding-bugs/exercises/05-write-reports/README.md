# Exercise 05 — Write Reports from Scratch

**Time:** ~35 minutes
**Grader:** structured-doc
**Pass threshold:** All 5 reports pass structural + keyword validation

## Your mission

You're given **5 bug descriptions** in plain English (no steps, no details). Write a complete bug report for each one from scratch.

You'll need to imagine the app context and write plausible, specific steps.

## Bug descriptions to report

1. "On the profile page, the bio text field doesn't save changes when you click Update"
2. "The search results page shows items from the wrong category"
3. "Password reset email never arrives after clicking the link"
4. "The date picker accepts February 30 as a valid date"
5. "After logging out, the back button shows the previous logged-in page"

## Deliverable

`starter/bugs.yaml` — 5 full bug reports.

Each report must include specific, actionable steps. Imagining a plausible app UI is fine.
