# React Fundamentals

Welcome to Part 3. You will learn React — the most widely used frontend library — by building components, managing state with hooks, fetching data, and creating forms.

## Learning Goals

- Understand JSX syntax and how it compiles to JavaScript
- Build reusable components that accept and render props
- Manage component state with `useState`
- Run side effects and fetch data with `useEffect`
- Build controlled form inputs
- Compose components into a working mini-application

## Exercises

| # | Slug | Type | Description |

| 4 | `04-controlled-forms` | test-runner | Controlled input forms |

## Prerequisites

- Part 1 and Part 2 complete
- Node.js 18+
- Familiarity with modern JavaScript (spread, destructuring, arrow functions)

## Testing Framework

Part 3 uses **Vitest** (instead of Jest) with **@testing-library/react**. Each exercise has:

```bash
cd starter
npm install
npm test
```
