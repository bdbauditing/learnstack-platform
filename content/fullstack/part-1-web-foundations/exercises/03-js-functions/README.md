# Exercise 03 — JavaScript Utility Functions

## Overview

Practice writing pure JavaScript functions using modern syntax: arrow functions, array methods, destructuring, and spread. These are the building blocks used throughout every part of the fullstack track.

## Your Task

Implement the functions in `starter/src/utils.js`. Each function has a `// TODO` comment explaining what it should do. Do **not** modify `__tests__/utils.test.js`.

## Functions to Implement

| Function | Description |
|----------|-------------|
| `sum(numbers)` | Return the sum of an array of numbers |
| `capitalize(str)` | Return the string with first letter uppercased |
| `uniqueValues(arr)` | Return array with duplicate values removed |
| `groupBy(arr, key)` | Group array of objects by a property key |
| `pick(obj, keys)` | Return new object with only the specified keys |
| `flattenOnce(arr)` | Flatten one level deep |
| `zipToObject(keys, values)` | Pair two arrays into one object |
