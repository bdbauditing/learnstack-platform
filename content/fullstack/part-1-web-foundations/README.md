# Web Foundations

Welcome to Part 1 of the LearnStack Fullstack Track. This part covers the fundamental building blocks of the web: how browsers and servers communicate, how HTML forms work, and how JavaScript powers modern applications.

## Learning Goals

By the end of this part you will be able to:

- Describe the full lifecycle of an HTTP request and response
- Explain the difference between GET and POST requests
- Design a basic REST API for a given resource
- Write and test pure JavaScript functions using modern syntax
- Use async/await to handle asynchronous operations
- Interpret network traffic in browser developer tools

## Prerequisites

- A modern web browser (Chrome or Firefox recommended)
- Node.js 18+ installed
- Basic familiarity with HTML and JavaScript syntax

## How to Work Through This Part

1. Read `concepts.md` thoroughly before starting exercises.
2. For **structured-doc** exercises, open `starter/submission.yaml` and fill in every field marked `TODO`.
3. For **test-runner** exercises, implement the functions in `starter/src/` and run `npm test` to verify.
4. Check the quiz when you finish all exercises.
